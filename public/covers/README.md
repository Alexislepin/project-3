# Placeholder Book Cover

Ce dossier contient le placeholder pour les couvertures de livres.

## Fichier requis

- `placeholder-book.png` - Image PNG du placeholder (à créer)

Un fichier SVG temporaire (`placeholder-book.svg`) est fourni comme référence.
Vous pouvez le convertir en PNG si nécessaire.

## Conversion SVG vers PNG

Si vous avez ImageMagick installé :
```bash
convert placeholder-book.svg placeholder-book.png
```

Ou utilisez un outil en ligne comme https://cloudconvert.com/svg-to-png
